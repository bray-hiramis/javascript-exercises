# The Interactive Todo / Task Manager

A simple interactive ToDo app!

## Features

1. Add new task
   - You can use the "Enter" key to add task.
2. Delete completed task

## Getting Started

You can clone this app with easy steps below:
```bash
git clone https://github.com/bray-hiramis/javascript-exercises.git
cd javascript-exercises
```

## Tech Stack

1. HTML5
2. CSS3
3. JavaScript
