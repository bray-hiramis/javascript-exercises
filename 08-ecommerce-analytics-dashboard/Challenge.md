# The Challenge: The E-Commerce Analytics Dashboard
You are building the backend analytics processor for an online store. You receive an array of recent customer order objects, and you need to use Higher-Order Functions to generate a summary report.

## Your Task
Write a function named <code>generateOrderSummary</code> that takes one parameter:

1. <code>orders</code> (an array of order objects)

Each order object looks like this:
```js
JavaScript
{ 
  orderId: "ORD-001", 
  customerName: "Alice", 
  items: [
    { name: "Laptop", price: 1000, category: "electronics" }, 
    { name: "Mouse", price: 50, category: "electronics" }
  ],
  isDelivered: true 
}
```
## The Rules
You must use JavaScript Higher-Order Array Methods (such as <code>.filter()</code>, <code>.map()</code>, <code>.reduce()</code>, or <code>.forEach()</code>) to process the data according to these steps:

1. <b>Filter</b>: Keep only the orders where <code>isDelivered</code> is <code>true</code> (we only want to analyze completed deliveries).

2. <b>Map/Transform</b>: For each of those completed orders, calculate the <b>total price</b> of all items inside its <code>items</code> array. Transform the filtered array so that instead of the raw order object, you return a clean summary object for each order, shaped like this:

    ```js
    { orderId: "ORD-001", customerName: "Alice", orderTotal: 1050 }
    ```

3. <b>Reduce</b>: Calculate the <b>grand total revenue</b> generated by all of those completed orders combined.

4. <b>Return</b>: The function should return a final summary object containing two properties:

    - <code>completedOrders</code>: The new array of transformed order summary objects you created in Step 2.

    - <code>totalRevenue</code>: The grand total revenue calculated in Step 3.

## Hints
- You can "chain" array methods together (e.g., <code>orders.filter(...).map(...)</code>) because <code>.filter()</code> and <code>.map()</code> both return brand-new arrays!

- Inside your <code>.map()</code> or when calculating prices, you can use <code>.reduce()</code> on the <code>items</code> array to quickly add up item prices.

- To get the <code>totalRevenue</code>, you can run a <code>.reduce()</code> on your newly mapped <code>completedOrders</code> array.
